## IT Job Salary Predictor

Проект по анализу данных и машинному обучению для прогнозирования зарплат в IT-сфере.  
Приложение позволяет предсказывать годовую зарплату специалиста на основе:

- должности
- категории профессии
- страны
- уровня опыта
- периода прогноза

Проект реализован как полноценное full-stack ML-приложение: от обработки данных и обучения модели до backend API и интерактивного frontend-интерфейса.

---

## О проекте

Данный проект предназначен для прогнозирования зарплат IT-специалистов в разных странах мира.

Пользователь выбирает:

- профессию
- направление деятельности
- страну
- уровень опыта
- количество лет прогноза

После этого модель машинного обучения рассчитывает предполагаемую зарплату, а интерфейс отображает результат вместе с графиком изменения зарплаты.

---

## Что было реализовано

В рамках проекта были выполнены следующие задачи:

- сбор и подготовка датасета с зарплатами IT-специалистов
- очистка и предобработка данных
- проведение exploratory data analysis (EDA)
- обучение ML-модели для прогнозирования зарплат
- сериализация модели и энкодеров через Pickle
- разработка backend API на Flask
- разработка frontend-интерфейса
- интеграция frontend и backend через REST API
- визуализация результатов и прогнозов

## Как работает приложение
1) Пользователь выбирает параметры в интерфейсе.
2) Frontend отправляет запрос в Flask API.
3) Backend кодирует категориальные признаки через сохранённые label encoders.
4) ML-модель рассчитывает прогноз зарплаты.
5) Результат возвращается на frontend.
6) Интерфейс отображает прогноз и график изменения зарплаты.


## ML Pipeline

Процесс подготовки и обучения модели разделён на несколько этапов:

1) 01_data_collection.ipynb - Сбор и подготовка данных.

2) 02_preprocessing.ipynb - Очистка и предобработка датасета.

3) 03_eda.ipynb - Exploratory Data Analysis:

   - исследование распределений
   - анализ зависимостей
   - визуализация данных

4) 04_model_training.ipynb - Обучение модели машинного обучения и сохранение итоговой модели.
